Module: Checkfront
Author: Checkfront Inc <support@checkfront.com>

Description
===========

This module connects your Wordpress blog to the Checkfront Onlibe Booking system.


Requirements
============

* Checkfront account http://www.checkfront.com

Installation
============
* Copy the 'checkfront' module directory in to your Drupal
sites/all/modules directory as usual.

Usage
=====
In the settings page enter your Checkfront URL.
